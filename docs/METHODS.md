# 方法说明：自监督去噪与 PSF 一致性反卷积

## 摘要

本项目面向显微图像中的微管结构恢复，目标是在缺少干净高分辨率标签的情况下，从有噪声、受光学点扩散函数模糊的观测图像中恢复更清晰、更细的结构。当前采用两阶段流程：第一阶段使用 N2V2 风格的盲点自监督网络对完整数据集进行去噪；第二阶段在去噪后的图像上训练反卷积网络，通过已知 PSF 的前向重投影约束，使网络输出经过成像模型后能够重新解释输入图像。

该流程的核心思想是把“噪声抑制”和“光学模糊逆问题”拆开处理。去噪模型负责降低随机噪声，避免反卷积阶段把噪声当作高频结构放大；反卷积模型负责恢复由 PSF 导致的空间扩散，并在正则项约束下尽量保留连续微管结构。

## 问题定义

显微图像可以近似表示为：

```text
x = D(h * y) + n
```

其中，`x` 是观测图像，`y` 是希望恢复的清晰结构，`h` 是 PSF，`*` 表示卷积，`D` 表示可能存在的采样或降采样过程，`n` 表示噪声。项目当前处理的是二维灰度 TIFF 图像，训练时通常将图像归一化到 `[0, 1]`。

如果直接从噪声图像训练反卷积模型，网络容易把噪声、背景起伏或局部亮度异常解释成真实高频结构，导致伪影、微管边缘复制、断裂或粗细不均。因此当前采用：

```text
有噪声观测图像
-> N2V2 自监督去噪
-> 去噪图像
-> PSF 约束反卷积 / 2x 超分辨率
-> 可选 rolling-ball 背景扣除
```

## 第一阶段：N2V2 自监督去噪

### 基本假设

N2V2 去噪依赖两个假设：

1. 噪声在空间上相对随机，单个像素处的噪声不能稳定地由邻域预测。
2. 真实微管结构具有空间相关性，连续线状结构可以由邻域上下文推断。

因此，如果训练时故意遮住一部分像素，让网络不能直接看到这些像素本身，而只能根据周围像素恢复它们，网络会倾向于学习空间连续结构，而不是记住随机噪声。

### 训练样本构造

对每个训练 patch，随机选择一小部分像素作为盲点。对于这些盲点位置，不把原始像素直接送入网络，而是用邻域中的其他像素替换它们，得到被扰动的输入图像。

流程可以表示为：

```text
原始 noisy patch
-> 随机选择 masked pixels
-> 用邻域像素替换 masked pixels
-> 得到 corrupted patch
-> 网络根据 corrupted patch 预测 denoised patch
-> 只在 masked pixels 上计算损失
```

损失函数为 masked L1：

```text
L_denoise = mean_mask |f(corrupted patch) - original patch|
```

这里的关键点是：损失只作用在被遮挡的像素上。由于这些像素在输入中已经被邻域值替换，网络无法通过恒等映射完成任务，只能从上下文中估计中心像素的合理值。

### 当前 N2V2 的作用

在本项目中，N2V2 的目的不是生成最终超分辨率图像，而是生成更适合反卷积训练的中间图像。它主要承担三件事：

1. 降低随机噪声，避免反卷积阶段放大噪声。
2. 平滑弱背景起伏，使 PSF 重投影 loss 更稳定。
3. 为多图反卷积训练提供更一致的输入分布。

但当前观察到，N2V2 也可能压低原本较亮的微管峰值，使残差图中出现微管结构。这说明去噪模型不只是去掉噪声，也在一定程度上改变了结构强度分布。这个问题会影响后续反卷积，使微管粗细和亮度不均匀。

## 第二阶段：PSF 一致性反卷积

### 反卷积目标

反卷积阶段希望学习一个映射：

```text
y_hat = g(x_denoised)
```

其中，`x_denoised` 是 N2V2 去噪后的图像，`y_hat` 是网络输出的清晰图像或 2x 超分辨率图像。由于没有真实高分辨率标签，不能直接让 `y_hat` 拟合 ground truth，而是使用物理前向模型约束它。

也就是说，网络输出 `y_hat` 后，需要经过 PSF 卷积重新变回观测域：

```text
x_reproj = h * y_hat
```

如果启用 2x 输出，则先在高分辨率网格上用调整后的 HR PSF 进行卷积，再降采样回输入大小：

```text
y_hat_hr
-> HR PSF 卷积
-> 降采样回输入尺寸
-> x_reproj
```

训练目标是让 `x_reproj` 尽量接近 `x_denoised`：

```text
L_reproj = |x_reproj - x_denoised|
```

这就是当前反卷积模型的核心约束：输出本身可以更尖锐、更高分辨率，但它经过 PSF 模糊后必须能解释原始观测。

### PSF 的处理原则

PSF 表示系统对点源的响应，必须保留其能量分布意义。因此当前项目中 PSF 只进行按和归一化：

```text
h = h / sum(h)
```

不应对 PSF 做 min-max 归一化或 percentile 归一化。否则 PSF 的能量守恒含义会被破坏，前向重投影不再对应真实成像模型。

PSF 中心当前更推荐使用图像中心而不是峰值中心，因为实际 PSF 文件可能已经按光学模型或采样坐标排列。若错误地按峰值重新裁剪或对齐，会造成前向模型偏移，表现为输出边缘复制、双边影或局部结构错位。

### 正则项

仅使用重投影损失会导致逆问题不适定。许多不同的 `y_hat` 经过 PSF 后都可能接近同一个 `x_denoised`，其中一部分解会含有伪影。因此当前加入了多个正则约束。

Hessian 正则约束二阶变化：

```text
L_hessian = mean((dxx y_hat)^2 + (dyy y_hat)^2 + (dxy y_hat)^2)
```

它的作用是抑制过强的局部振荡，减少针状噪声和边缘复制。但权重过大时会使微管变粗、变钝，损失高频细节。

TV 正则约束一阶变化：

```text
L_tv = mean((dx y_hat)^2 + (dy y_hat)^2)
```

它可以抑制孤立噪点和碎片化纹理，但过强时会让连续微管局部被抹平，细节变少。

部分实验中还使用 detail branch，即在主干输出基础上增加一个受控高频残差：

```text
y_hat = y_base + alpha * y_detail
```

该分支可以让微管更细，恢复更多高频结构。但如果约束不足，残差分支也容易制造伪影，例如主微管两侧的复制结构、断裂纹理或不真实的尖锐边缘。因此当前更倾向于使用较小的 `detail_scale`，并配合 detail L1、Hessian 和 TV 限制其幅度。

### 总体损失

当前反卷积训练目标可以概括为：

```text
L_deconv =
  L_reproj
  + lambda_hessian * L_hessian
  + lambda_tv * L_tv
  + lambda_l1 * L_intensity
  + lambda_detail * L_detail
  + lambda_grad * L_gradient_reproj
```

其中核心项是 `L_reproj`。其他项用于选择更合理、更平滑、更连续的解。

## 推理流程

完整推理可分为两步：

1. 使用 N2V2 模型对原始噪声 TIFF 去噪，得到 `x_denoised`。
2. 使用反卷积模型对 `x_denoised` 推理，得到更清晰或 2x 尺寸的 `y_hat`。

反卷积推理时通常使用 reflect padding，避免边界卷积产生强烈边缘伪影。输出后可选 rolling-ball 背景扣除，用于压低缓慢变化的背景。需要注意，rolling-ball 背景扣除是后处理，不改变模型本身；它可以改善视觉背景，但不能恢复被模型抹掉的结构，也可能影响强度定量。

## 当前现象与原因分析

### 去噪后反卷积不如无噪声样例稳定

目前观察到，噪声数据经过 N2V2 再反卷积后，微管结构可能出现粗细不均匀。主要原因可能是：

1. N2V2 在去噪时压低了亮微管峰值，改变了横截面强度。
2. 去噪残差中仍含有结构，说明部分真实高频被当作噪声削弱。
3. 反卷积阶段会进一步放大输入中的横截面差异。
4. 多图数据中的曝光、背景、噪声强度不一致，使模型学习到折中的恢复策略。

### 残差分支可以变细，但容易产生伪影

残差或 detail branch 提供了额外的高频自由度，因此能让微管更细。然而反卷积本身是病态逆问题，高频自由度越大，越容易把不确定的成分解释成结构。出现微管两侧复制、局部断裂和纹理伪影，通常说明高频增强超过了物理重投影约束和连续性正则的控制能力。

### 单图训练更容易过拟合局部纹理

单图 zero-shot 训练只有一张图像提供统计信息，网络容易学习该图中的局部噪声、背景纹理或偶然结构。多图训练能够提供更丰富的形态变化，有助于抑制特定图像上的伪影。但多图训练的前提是数据分布和归一化足够稳定，否则也可能引入粗细不均匀。

## 潜在改进方向

### 1. 改进 N2V2，减少结构误删

当前最重要的问题之一是 N2V2 残差中存在微管结构。可以从以下方向改进：

1. 降低 mask ratio，减少训练时被替换像素比例，避免网络过度依赖平滑上下文。
2. 调整邻域大小，避免用距离太近或太远的像素替换中心点造成结构偏置。
3. 引入结构保持损失，例如梯度一致性或 Hessian 一致性，使去噪前后的线状结构横截面不被明显削弱。
4. 使用 Noise2Void 后处理策略，例如多次随机 mask 推理平均，降低单次盲点采样造成的偏差。
5. 对强结构区域降低去噪强度，对背景区域提高去噪强度，形成结构自适应去噪。

### 2. 反卷积训练中加入结构连续性约束

微管是连续线状结构，不是孤立点状纹理。后续可以加入更符合微管形态的先验：

1. Frangi vesselness 或 Hessian-based line prior，鼓励线状连续结构。
2. 方向一致性约束，使局部线段方向变化更平滑。
3. Skeleton 或 ridge consistency，用于惩罚断裂和双边复制。
4. 多尺度重投影损失，让粗尺度结构和细尺度边缘同时满足物理一致性。

### 3. 控制高频残差分支

detail branch 当前能提升锐度，但也带来伪影。可尝试：

1. 降低 `detail_scale`，让残差只做轻微补偿。
2. 增大 `detail_l1_w`，限制残差幅度。
3. 只允许残差出现在高置信结构区域，例如根据输入梯度或 vesselness 生成 gate。
4. 对残差分支加入频域约束，避免生成与 PSF 不一致的周期性复制纹理。
5. 分阶段训练：先关闭 detail branch 训练稳定主干，再短时间微调 detail branch。

### 4. 使用更准确的噪声模型

显微图像噪声通常不是简单高斯噪声，可能包含 Poisson shot noise、读出噪声和背景漂移。后续可以：

1. 使用 Poisson-Gaussian 噪声建模。
2. 在训练 patch 中按亮度自适应加入噪声扰动。
3. 对不同图像估计背景和噪声水平，进行分组训练或条件输入。
4. 在去噪阶段保留强度校准，避免不同图像 percentile normalization 后失去定量关系。

### 5. 改进 PSF 与前向模型

当前反卷积质量高度依赖 PSF 是否正确。可以考虑：

1. 检查 PSF 尺寸、中心和采样间距是否与图像一致。
2. 对 2x 输出使用更严格的 HR PSF 构造，而不是简单插值。
3. 在固定 PSF 基础上学习一个小的 PSF 修正项，但要保持非负和归一化。
4. 同时约束图像域重投影和梯度域重投影，减少模糊一致但边缘错误的解。

### 6. 引入跨图像验证指标

当前主要依赖视觉观察，后续应建立更稳定的评估方式：

1. 对去噪残差做结构检测，量化残差中的微管成分。
2. 测量微管横截面 FWHM 或线宽分布，判断是否粗细不均。
3. 比较重投影图与输入图的误差图，检查是否存在系统性偏移或双边残差。
4. 使用同一模型推理多张图，观察结构连续性、背景一致性和伪影重复模式。

## 推荐的实验路线

短期内建议优先做三组实验：

1. 优化 N2V2，使残差图中尽量不再保留明显微管结构。
2. 使用较保守的反卷积主干训练一个真实感更好的 baseline。
3. 在 baseline 上小幅加入 detail branch，并逐步调节 `detail_scale` 和 `detail_l1_w`，寻找“更细”和“更真实”之间的平衡点。

更长期的方向是把去噪、反卷积和结构先验统一起来，但不建议过早做端到端联合训练。当前更稳妥的路线是先把每个阶段的误差来源分开看清楚：去噪是否误删结构、PSF 前向是否一致、反卷积是否过度使用高频自由度。只有这些问题分别稳定后，联合优化才更有意义。
